# GEF — Site Vitrine

Site web officiel du **Guardian Engineering Framework**, construit avec React + TypeScript + Vite.

## Lancer en local

```bash
cd website
npm install
npm run dev
```

## Contribuer

Toute modification doit passer par une Pull Request sur la branche `main` du dépôt GEF.
Les changements sur le site vitrine sont soumis aux mêmes règles que le reste du framework.
